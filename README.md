# Scruf's Scene Collection

Auto-generated Foundry VTT scenes created by Jr (AntiScruffy Jr.), the digital familiar.

## Installation

1. In Foundry VTT: **Add-on Modules → Install Module**
2. Paste manifest URL: `https://raw.githubusercontent.com/Scrufffington/Foundry/main/module.json`
3. Click Install
4. Enable module in your world (**Game Settings → Manage Modules**)

## Building the Compendium (IMPORTANT)

The scenes are stored as JSON files. To use them in Foundry, you need to build a LevelDB compendium.

### Option 1: Quick Build Script (Recommended)

SSH into your Foundry server and run:

```bash
cd Data/modules/scruf-scenes/scripts
./build-compendium.sh
```

This will:
- Install the `level` npm package if needed
- Convert JSON scenes to LevelDB format
- Create the compendium in `packs/scenes.db/`

Then restart Foundry.

### Option 2: Manual Build

```bash
cd Data/modules/scruf-scenes/scripts
npm install level
node build-compendium-level.js
```

Restart Foundry after building.

## Using the Scenes

After building the compendium:

1. **Open Compendium Packs** (book icon on right sidebar)
2. Look for **"Jr's Scene Collection"**
3. Import scenes:
   - **Drag to Scenes tab** to add to your world
   - **Right-click → Import** to import individually
   - **Right-click → Import All** to import everything

## Getting Updates

When new scenes are added:
1. In Foundry: **Add-on Modules → Check for Updates**
2. Click Update on "Scruf's Scene Collection"
3. Re-run the build script: `./build-compendium.sh`
4. Restart Foundry

## For Scene Developers

To generate new scenes:

```bash
./scripts/generate-and-commit-scene.sh "Scene Name" "Description"
```

Scenes are automatically pushed to this repo.
