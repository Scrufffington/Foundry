# Scruf's Scene Collection

Auto-generated Foundry VTT scenes created by Jr (AntiScruffy Jr.), the digital familiar.

## Installation

1. In Foundry VTT: **Add-on Modules → Install Module**
2. Paste manifest URL: `https://raw.githubusercontent.com/Scrufffington/Foundry/main/module.json`
3. Click Install
4. Enable module in your world (**Game Settings → Manage Modules**)

## Accessing Scenes

After installing and enabling the module, scenes are located at:
```
Data/modules/scruf-scenes/scenes/
```

### Method 1: Scene Packer Module (Recommended)
If you have the **Scene Packer** module installed:
1. Go to **Settings → Module Configuration → Scene Packer**
2. Import scenes from "Scruf's Scene Collection"

### Method 2: Manual Import
1. Open your world's Scenes tab
2. Click **Create Scene**
3. Use **Import from JSON** and browse to:
   ```
   Data/modules/scruf-scenes/scenes/dark-cathedral.json
   ```

### Method 3: Copy Files
Copy scene JSON files from the module to your world:
```bash
cp Data/modules/scruf-scenes/scenes/*.json \
   Data/worlds/YOUR-WORLD/scenes/
```
Then restart Foundry.

## Available Scenes

- **Dark Cathedral** - Gothic cathedral with demon altar
- **Goblin's Grotto** - Putrid goblin bathing grotto

## Generating New Scenes

New scenes are automatically generated and pushed to this repository. To get updates:
1. In Foundry: **Add-on Modules → Check for Updates**
2. Or reinstall from the manifest URL
