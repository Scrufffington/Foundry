# Scruf's Scene Collection

Auto-generated Foundry VTT scenes created by Jr (AntiScruffy Jr.), the digital familiar.

## Installation

1. In Foundry VTT: Add-on Modules → Install Module
2. Paste manifest URL: `https://raw.githubusercontent.com/Scrufffington/Foundry/main/module.json`
3. Enable module in your world
4. Scenes will be available in the Scenes directory

## Scenes

- `dark-cathedral.json` - Gothic cathedral with demon altar
- `goblin's-grotto.json` - Putrid goblin bathing grotto

## Usage

Import scenes into your world using the Scene Packer module, or copy JSON files to your `Data/worlds/YOUR-WORLD/scenes/` folder.
